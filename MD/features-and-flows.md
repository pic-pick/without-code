### 2-1. 회원가입 / 로그인 플로우

```
[앱 실행]
    │
    ▼
[세션 토큰 확인]
    │
    ├─ 유효 ──────────────────────► [메인 피드]
    │
    └─ 없음 / 만료
            │
            ▼
        [랜딩 / 로그인 화면]
            │
            ├─ [카카오로 계속하기]
            ├─ [구글로 계속하기]
            └─ [GitHub로 계속하기]
                    │
                    ▼
            [OAuth 인증 처리]
                    │
                    ├─ 실패 ─► [오류 토스트] ─► [로그인 화면으로]
                    │
                    └─ 성공
                            │
                            ├─ 기존 유저 ─────────► [메인 피드]
                            │
                            └─ 신규 유저
                                    │
                                    ▼
                            [온보딩 플로우 (2-2)]
```

```
        │                              │

        ▼                              │

        ▼
```

### 2-3. 프로젝트 쇼케이스 업로드 플로우

```
[하단 탭 + 버튼 탭]
        │
        ├─ 비로그인 ─► [로그인 유도 바텀 시트] ─► [로그인 화면]
        │
        └─ 로그인 상태
                │
                ▼
        [게시글 유형 선택]
            │          │
      [쇼케이스]    [자유 게시글]
            │
            ▼
    [쇼케이스 업로드 폼]
        ┌───────────────────────┐
        │ 1. 썸네일 이미지 선택  │  ← 갤러리 / 카메라
        │ 2. 제목 입력 (필수)    │  ← 최대 60자
        │ 3. 설명 입력           │  ← 최대 500자
        │ 4. 데모 링크 (필수)    │  ← https:// 검증
        │ 5. GitHub 링크 (선택)  │
        │ 6. 툴 태그 선택        │  ← 최대 5개
        │ 7. 프롬프트 입력+공개  │  ← Phase 3
        └───────────────────────┘
                │
                ▼
        [유효성 검사]
                │
                ├─ 실패 ─► [인라인 에러 메시지] ─► 폼으로
                │
                └─ 통과
                        │
                        ▼
                [이미지 업로드 (프로그레스)]
                        │
                        ├─ 실패 ─► [재시도 토스트]
                        │
```
